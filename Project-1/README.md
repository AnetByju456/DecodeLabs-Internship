# TradeMart

TradeMart is a responsive frontend marketplace platform that enables users to buy, sell, borrow, and lend products through a unified and user-friendly experience.

The project focuses on creating a modern, accessible, and responsive marketplace interface while demonstrating frontend development concepts such as authentication flows, product management, filtering, responsive design, and role-based user experiences.

Developed as part of an internship project, TradeMart is designed with future full-stack expansion in mind.

---

## Project Overview

TradeMart provides a single platform where users can:

* Buy products from other users
* Sell products through listings
* Borrow items when needed
* Lend products to other users

The platform supports different user roles and delivers a tailored experience based on the selected role.
